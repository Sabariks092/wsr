# Wild South Raiders — Complete Design Language System

Based on the visual DNA of the provided artwork, the brand aesthetic can be described as:

> **Rugged Western + Vintage Outlaw + Burnt Desert + Cinematic Adventure**

The design language should feel:

* Bold
* Fearless
* Dusty
* Weathered
* Vintage
* Handcrafted
* Cinematic
* High Contrast
* Masculine
* Adventure-driven

---

# 1. Core Brand Personality

### Keywords

* Outlaw
* Frontier
* Desert
* Gunslinger
* Dust & Smoke
* Rustic
* Vintage Wood

* Survival

| Attribute | Scale |

# 3. Extended UI Palette

```css
--success: #5C8A3D;
--warning: #D88A2D;
--danger: #B62323;
--info: #B58A4A;
```

All colors should feel dusty and earthy.

Avoid:

❌ Neon colors
❌ Pure blue
❌ Bright green
❌ Tech-style gradients

---

# 4. Typography System

The logo typography resembles:

### Primary Display Font

Best matches:

* Rye
* Alfred Stout
* Playbill
* Black Ops One
* Sancreek

Recommended:

```css
font-family: "Rye", serif;
```

Used for:

* Hero headings
* Banner titles
* Section titles

---

### Secondary Font

For readable content:

```css
font-family: "Poppins", sans-serif;
```

Weights:

```text
400 Regular
500 Medium
600 SemiBold
700 Bold
```

# 5. Texture Language

The artwork is NOT flat.

Everything has:

### Required

* Dust texture
* Scratches
* Grain
* Worn paint
* Leather feel
* Aged wood feel

### Avoid

* Glassmorphism
* Frosted effects
* Modern SaaS aesthetics
* Pure flat UI

---

### Primary Button

Background:

```css

```


Border:


# 11. Shadows & Depth

Heavy cinematic shadows.

```css
box-shadow:
0 10px 30px rgba(0,0,0,.35);
```

Text shadows:

```css
2px 4px 8px rgba(0,0,0,.45);
```

---

# 12. Border Language

No sharp corporate borders.

Use:

```css
border-radius: 12px;
```

or

```css
border-radius: 18px;
```

Decorative gold borders preferred.

---

# 13. Animation System

Animations should feel subtle and atmospheric.

Allowed:

* Dust floating
* Ember particles
* Slow parallax
* Smoke movement
* Horse idle movement

Duration:

```css
2s–8s
```

Avoid:

❌ Bounce
❌ Flashing
❌ Neon glows
❌ Fast zooms

# Final Visual Direction

If you hand this to a developer, the guiding rule should be:

> **"Build it like a premium Western adventure game website, not a modern SaaS dashboard. Every component should feel aged, dusty, cinematic, handcrafted, and rugged while maintaining readability and usability."**
